# RFC 0001: SIP-Core Public Draft

## Summary

Define a minimal public manifest profile for Schema Sandbox-compatible capability packages.

## Motivation

A minimal profile makes the methodology easier to cite, discuss, and implement without requiring a complete production runtime.

## Draft fields

- metadata
- input_contract
- output_contract
- permission_scope
- error_codes
- optional validation
- optional workspace_partitions

## Status

Open draft.
