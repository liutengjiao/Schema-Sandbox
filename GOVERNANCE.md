# Governance

Schema Sandbox is currently governed as an author-led public methodology draft.

## Versioning

- `v0.x`: Public methodology and draft specification stage
- `v1.0`: Stable public methodology release
- `SIP-Core`: Minimal interoperable manifest profile
- `Industrial Middleware Profile`: Reference profile for industrial AI agent execution gateways

## Change policy

The project prioritizes:

1. Clear positioning
2. Citation stability
3. Safe reference examples
4. Implementation neutrality
5. Explicit limits and disclaimers

Breaking changes to public field names, license language, attribution requirements, or terminology should be introduced through an RFC document under `rfc/`.
