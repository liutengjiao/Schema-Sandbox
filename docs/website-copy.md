# SchemaSandbox.com Website Copy Draft

## Home Hero

**Schema Sandbox**

The contract layer between probabilistic LLMs and deterministic industrial execution.

Schema Sandbox defines a public methodology and interoperability contract for validating, permissioning, isolating, and auditing AI agent actions before they reach enterprise or industrial systems.

Buttons:

- Read the Methodology

## Four homepage blocks

### For AI Agents

Validate actions before tools are called.

### For Industrial Workflows

Place a contract layer before MES, ERP, QMS, CMMS, EHS, and work-order systems.

### For Developers

Use SIP manifests, input/output contracts, permission scopes, rejection payloads, and evidence records.

### For Researchers

Cite the public methodology and contribute to the open specification.

## Industrial page hero

**Industrial AI Middleware Reference Patterns**

AI agents should not directly touch critical industrial systems. Schema Sandbox shows how to place a contract layer between agent proposals and MES, ERP, QMS, CMMS, EHS, databases, and internal APIs.
