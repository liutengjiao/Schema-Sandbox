# ERP / MES Permission Gateway

This directory contains a reference manifest and example payloads for the `erp_mes_permission_gateway` sandbox pattern.

Status: reference example only. Not a production implementation.

Files:

- `manifest.json`: SIP-Core reference manifest
- `sample-request.json`: example proposed action
- `sample-response-allow.json`: example allowed response
- `sample-response-reject.json`: example rejection payload
